System Overview

What JailWatch PH Does

A machine learning–powered web application that analyzes BJMP regional jail data to surface hidden patterns and risk profiles across the Philippines.

Data Consolidation

Eight BJMP statistical tables — jail population, crimes charged, congestion rates, drug cases, age profiles, and PDL flow — merged into one unified dataset of 18 regions × 30 features.

View Methodology →

K-Means Clustering

Using Python's scikit-learn, regions are clustered into 4 risk groups based on 6 key features: total PDL, congestion rate, drug cases, total crimes, violence indicators, and PDL flow.

Dataset

30 Features Across 7 Categories

All features extracted from official BJMP statistical reports as of July 31, 2025.

Jail Population 6 features
  • Total PDL (Persons Deprived of Liberty)
  • Detainee Male / Female
  • Sentenced Male / Female
  • Number of Jails per Region
Jail Infrastructure 3 features
  • Number of District Jails
  • Number of City Jails
  • Number of Municipal Jails
Crimes Charged 7 features
  • Total Crimes (Index + Non-Index)
  • Murder · Homicide · Rape
  • Physical Injury
  • Robbery · Theft
Congestion 4 features
  • Average Congestion Rate (%)
  • Ideal Capacity
  • Actual PDL Population
  • Number of Congested Jails
PDL Flow 2 features
  • Monthly PDL Committed
  • Monthly PDL Released
Drug Cases 3 features
  • Total PDL with Drug Cases
  • Sentenced Drug PDL
  • Awaiting Trial Drug PDL
Age Profile 4 features
  • Below 18 (Minors)
  • Age 18–30
  • Age 31–41
  • Above 42
